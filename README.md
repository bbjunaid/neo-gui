Syncer

Install:
- cygwin
- cygwin git, wget, curl, python


`python run_sync.py`

Commands
===
`./restart_sync.sh`
All syncers will restart themselves once.
`./stop_restart_sync.sh`
Stop sending the restart signal everytime you push.
